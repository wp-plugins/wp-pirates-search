== Description ==

Plug-in automatically interrogates the search engines to find copies of articles and blog entries from your own site. Found copyes are displayed in a list of links. You can disable the search for copies of individual articles. Found copyes can be marked as a legal copy. You can also specify sites whose pages are always going to be excluded from the search.

Default settings are sufficient for 99% of sites.

== Installation ==

1. Upload full directory into your wp-content/plugins directory
2. Activate the plugin at plugin administration page
3. Open the plugin configuration page, which is located under *Options > Search for pirates* and check settings.
